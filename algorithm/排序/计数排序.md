# 计数排序

#### 基本假设:

$$n$$个输入元素中的每一个都是0到$$k$$区间内的一个整数，其中$$k$$为某个整数。

#### 基本思想：

对每一个输入元素$$x$$，确定小于$$x$$的元素个数，利用这一信息，直接将$$x$$放到输出数组中的位置上。

#### 实现

```go
// CountSort O(n + k)
func CountSort(A []int, k int) {
	c := make([]int, k+1)
	for _, a := range A {
		c[a]++
	}
	for i := 1; i < k+1; i++ {
		c[i] += c[i-1]
	}
	b := make([]int, len(A))
    // 从后往前，保证排序的稳定
	for j := len(A) - 1; j >= 0; j-- {
		c[A[j]]--
		b[c[A[j]]] = A[j]
	}
	for i := range A {
		A[i] = b[i]
	}
}
```
