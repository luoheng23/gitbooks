# 简记法

| 转义符 | 含义                   |
| ------ | ---------------------- |
| \t     | 制表符                 |
| \n     | 换行符                 |
| \r     | 回车符                 |
| \s     | 空白字符（空格，制表） |
| \S     | \s取反                 |
| \w     | [a-zA-Z0-9]            |
| \W     | \w取反                 |
| \d     | [0-9]                  |
| \D     | \d取反                 |
