# 顺序统计树

顺序统计树是[红黑树](红黑树.md)的变种。

顺序统计树在红黑树的基础上，给每个结点添加了$$size$$属性，$$x.size = x.left.size + x.right.size + 1$$。因此可以在$$O(lgn)$$的时间内确定任何的顺序统计量。

### 实现

```go
type color bool

const (
	black color = true
	red   color = false
)

// StatNode of tree
type StatNode struct {
	key         int
	color       color
    // 增加一个size属性
	size        int
	left, right *StatNode
	p           *StatNode
}

// StatTree is black and red tree
type StatTree struct {
	null *StatNode
	root *StatNode
}

// CreateStatTree create a tree
func CreateStatTree(A []int) StatTree {
	r := StatTree{
		null: &StatNode{color: black, size: 0},
	}
	r.root = r.null
	for _, a := range A {
		r.Insert(&StatNode{key: a})
	}
	return r
}
```

### 树的旋转

与红黑树的一致，在最后面添加两行代码以维持$$size$$属性。

#### 左旋

```go
func (r *StatTree) leftRotate(x *StatNode) {
	y := x.right
	x.right = y.left
	if y.left != r.null {
		y.left.p = x
	}
	y.p = x.p
	if x.p == r.null {
		r.root = y
	} else if x == x.p.left {
		x.p.left = y
	} else {
		x.p.right = y
	}
	y.left = x
	x.p = y
    // 维护size
	y.size = x.size
	x.size = x.left.size + x.right.size + 1
}
```

#### 右旋

```go
func (r *StatTree) rightRotate(x *StatNode) {
	y := x.left
	x.left = y.right
	if y.right != r.null {
		y.left.p = x
	}
	y.p = x.p
	if x.p == r.null {
		r.root = y
	} else if x == x.p.left {
		x.p.left = y
	} else {
		x.p.right = y
	}
	y.right = x
	x.p = y
    // 维护size
	y.size = x.size
	x.size = x.left.size + x.right.size + 1
}
```

### 结点插入

插入结点的祖先节点的$$size$$都减1。

```go
// Insert a StatNode
func (r *StatTree) Insert(z *StatNode) {
	y, x := r.null, r.root
	for x != r.null {
		y = x
		if z.key < x.key {
			x = x.left
		} else {
			x = x.right
		}
	}
	z.p = y
	if y == r.null {
		r.root = z
	} else if z.key < y.key {
		y.left = z
	} else {
		y.right = z
	}
	z.left = r.null
	z.right = r.null
	z.color = red
	z.size = 1
    // 从插入结点向上，所有祖先结点的size-1
	x = z.p
	for x != r.null {
		x.size++
		x = x.p
	}
	r.insertFixUp(z)
}
```

### 结点删除

删除的或者移动的结点的祖先结点$$size$$减1。

```go
// Delete a StatNode
func (r *StatTree) Delete(z *StatNode) {
	y, yColor := z, z.color
	x := z
	if z.left == r.null {
		x = z.right
		r.transplant(z, z.right)
	} else if z.right == r.null {
		x = z.left
		r.transplant(z, z.left)
	} else {
		y = r.Min(z.right)
		yColor = y.color
		x = y.right
		if y.p == z {
			x.p = y
		} else {
			r.transplant(y, y.right)
			y.right = z.right
			y.right.p = y
		}
		r.transplant(z, y)
		y.left = z.left
		y.left.p = y
		y.color = z.color
		y.size = z.size
	}
    // 删除或移动结点的所有祖先结点size-1
	t := x.p
	for t != r.null {
		t.size--
		t = t.p
	}
	if yColor == black {
		r.deleteFixUp(x)
	}
}
```

### 查找第i个顺序统计量

```go
// Select the StatNode which is ith node O(lgn)
func (r *StatTree) Select(x *StatNode, i int) *StatNode {
	s := x.left.size + 1
	if i == s {
		return x
	} else if i < s {
		return r.Select(x.left, i)
	} else {
		return r.Select(x.right, i-s)
	}
}
```

### 判断当前结点是第几个顺序统计量

```go
// Rank the x node
func (r *StatTree) Rank(x *StatNode) int {
	s := x.left.size + 1
	y := x
	for y != r.root {
		if y == y.p.right {
			s += y.p.left.size + 1
		}
		y = y.p
	}
	return s
}
```
