

## 字符串

字符串不可更改。

```go
s := "hello, world"
len(s)  // 12
s[0], s[7] // 'h'  'w'
s[0:5] // hello
s[:]   // hello, world
```

```go
"hello, world" // 字符串字面量
`hello` // 原生字符串字面量(不含转义序列，可实现多行字符串)
```

## 常量

```go
const pi = 3.14159
const (
	a = 1
    b     // 1
    c = 2
    d     // 2
)
const (
	a = iota // 0
    b        // 1
    c        // 2
    d = 2
    e = iota // 4
)
```

## 无类型常量

无类型常量的精度更高，而且无需转换类型。
