# 汇编语言

## 用途

* **嵌入式系统**

* 实时应用
* 驱动程序

## 与高级语言对比

| 应用程序类型   | 高级语言                             | 汇编语言                       |
| -------------- | ------------------------------------ | ------------------------------ |
| 中大型商业软件 | 维护方便                             | 维护困难                       |
| 硬件设备驱动   | 维护困难，没有提供直接访问硬件的能力 | 程序短时容易维护，容易访问硬件 |
| 多平台商业软件 | 方便移植                             | 不好移植，语法各不相同         |
