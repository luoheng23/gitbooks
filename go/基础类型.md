# 基础类型

## 整数

```go
int int8 int16 int32 int64
uint uint8 uint16 uint32 uint64
```

```go
t := 0 // int
```

## 浮点数

```go
float32 float64
```

```go
f := 1e100
var z float32 = 166666
var s float64 = 1.23234234
```

## 复数

```go
complex64 // float32
complex128 // float64
```

```go
var x complex128 = complex(1, 2)
x := 1 + 2i
var y complex128 = complex(3, 4)
y := 3 + 4i
```

## 布尔值

```go
true false
```

布尔值与数值无法隐式转换！！

## 字符串

字符串不可更改。

```go
s := "hello, world"
len(s)  // 12
s[0], s[7] // 'h'  'w'
s[0:5] // hello
s[:]   // hello, world
```

```go
"hello, world" // 字符串字面量
`hello` // 原生字符串字面量(不含转义序列，可实现多行字符串)
```

## 常量

```go
const pi = 3.14159
const (
	a = 1
    b     // 1
    c = 2
    d     // 2
)
const (
	a = iota // 0
    b        // 1
    c        // 2
    d = 2
    e = iota // 4
)
```

## 无类型常量

无类型常量的精度更高，而且无需转换类型。
