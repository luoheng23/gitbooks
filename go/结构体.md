# 结构体

## 定义

```go
type Employee struct {
    ID int
    Name string
    Address string
    DoB time.Time
    Position string
    Salary int
    ManagerID int
}
```

## 成员访问

不可导出的变量在包外无法使用。

```go
var dilbert Employee
dilbert.Salary = 5000
```

## 初始化

```go
type Point struct {
    X, Y int
}

a := Point{1, 2}
b := Point{X: 1, Y: 2}
```

## 比较

如果结构体的所有变量都可以比较，那么这个结构体就是可比较的。

## 结构体嵌套与匿名成员

```go
type Point struct {
    X, Y int
}

type Circle struct {
    Point
    Radius int
}

type Wheel struct {
    Circle
    Spokes int
}
```
